# App web React + Vite

Cette application contient :

- l'application produit pour les utilisateurs authentifies
- le back-office admin interne sous `/admin`

## Commandes utiles

Depuis `apps/web` :

```bash
pnpm dev
pnpm build
pnpm lint
pnpm test:e2e
```
